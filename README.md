# Projekt: Modelowanie Deterministyczne - Równanie Ciepła

Projekt dotyczy numerycznego rozwiązania równania ciepła w celu analizy efektywności ogrzewania pomieszczeń. Symulacja sprawdza m.in. wpływ lokalizacji grzejnika na komfort cieplny oraz zjawisko pasożytnictwa cieplnego.

## Struktura projektu
- `data`: Parametry fizyczne materiałów, moce grzejników i dane pogodowe.
- `pipeline`: Moduły Pythonowe zawierające logikę obliczeniową (klasy i funkcje).
- `notebooks`: Notatniki Jupyter z przebiegiem eksperymentów i wizualizacjami.
- `requirements.txt`: Lista bibliotek potrzebnych do uruchomienia projektu.

## Jak uruchomić?
1. Zainstaluj wymagane biblioteki: `pip install -r requirements.txt`
2. Otwórz wybrany notatnik w folderze `notebooks`.
